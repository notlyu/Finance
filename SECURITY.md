## Medium Issues

### 3. CORS Config

**File:** `server.js:32-40`

**Code:**
```js
app.use(cors({
  origin: corsOrigins.length ? corsOrigins : true,  // Dangerous!
  credentials: process.env.CORS_CREDENTIALS === 'true',
}));
```

**Risk:** If `CORS_ORIGINS` is empty, `origin: true` allows all

**Recommendation:**
```js
const corsOrigins = process.env.CORS_ORIGINS?.split(',').filter(Boolean) || [];
if (corsOrigins.length === 0) {
  throw new Error('CORS_ORIGINS must be set');
}
app.use(cors({
  origin: corsOrigins,
  credentials: true,
}));
```
